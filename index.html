Web Design Homework - Web Visualization Dashboard (Latitude)

Latitude Analysis Dashboard with Attitude

The purpose of this project was to analyze howweather changes as you get closer to the equator. To accomplish that we first pulled data-page from the OenWeatherMap API to assemble a database on over 500 cities.

After assembling the dataset, we used matplotlib to plot various aspects of the weather vs latitude. Factors we looked at included: temperature, cloudiness, wind-speed and humidity. This site provides the source data-page and visualizations created as part of the analysis, as well as explanations description of any trends and corelation witnessed.
